O pedido

Liberar o acesso à API (allowlist)

  • As rotas GET /api/worlds e GET /api/characters/search funcionam no navegador, mas o Cloudflare bloqueia o acesso do nosso servidor.
  • Bastaria liberar o User-Agent RubibotStatistics/1.0 — ou orientar sobre uma API oficial para fansites.
  • Consumo responsável: feliz em adotar cache e limites de frequência que vocês definirem.
